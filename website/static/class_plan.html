教材部分

1. 課程教材編排以「人」為主軸，擴及自我與社會他人以及自然環境的關係。

2.教材內容規劃為「親情」、「友情與愛情」、「生命關懷」、「國族與社會關懷」、「環境關懷」等五大單元，前三個單元以個人生命歷程為主，由親情、友情、愛情，進而省察個人生命意義與價值；後二個單元則擴及至 關懷他人(國族與社會)，以及關懷自然環境。

3. 每一主題單元閱讀規劃精讀古典文學、現代文學至少三至四篇，以及指定延伸閱讀文章三至四篇，以多樣體裁的古今文學作品相融合，培養學習者閱讀興趣及提升語文運用能力，並對該主題有較深入的探求與認知。

4. 課程教材現代文學部分扣合學校所在環境與學生來源，納入客家、原住民族的相關文章閱讀。

5. 每一篇文章皆有「作者」、「導讀」、「本文」、「注釋」、「語譯」、「議題討論」等內容以供閱讀討論。另外，除了教材選文外，每一單元後皆新增「單元延伸閱讀」，提供與單元主題相關之課外書籍文章與影音資料，教師可配合運用，做為學生課後閱讀之篇章。

作業部分

1. 運用每一篇文章的「議題討論」進行分組討論，訓練學習者對相關議題的溝通、探討能力。

2. 每學期配合單元主題擬定相關題目，使學習者進行「主題書寫」兩篇，以訓練其邏輯思考、組織與表達能力，並擴展其寫作層面。此書寫由任課老師針對各主題單元命題，要求每位學習者進行寫作，再由老師收回批閱。

3. 每一單元皆有設計「主題單元學習單」，使學習者進行小組討論報告與寫作，以提升學習者組織能力、書寫能力、口語表達能力，並培養分工合作的團隊精神。此學習單每份設計三個議題，經由各小組討論後寫作，再交給與任課老師分別進行回應。

4. 規劃「校外參訪學習單」使學習者書寫、紀錄校外的參訪或參與相關活動。

5. 每學期配合主題單元邀請專業人士進行2場「專題講座」，使學習者撰寫講座內容摘要與感想，以啟迪其更寬廣的閱聽視野。所謂專業人士，不侷限於純文學作家，同時照應作家與其他社會領域人士，如媒體新聞工作者、藝文界人士、環保工作者、社會工作者等相關領域。
